## 즐농(농구동아리 관리 서비스)

----------
## 즐농 기능 설명
1. 클럽 생성 가능(최대 2개)
   1. 클럽 정보(클럽명, 등급, 마크)
2. 클럽 가입 가능(최대 5개)
3. 클럽 가입 요청
4. 클럽 가입 승인
5. 게임 생성 가능 (체육관 설정시에만 가능)
   1. 자체전 : 클럽 성적엔 반영 안됌.
   2. 교류전 : 클럽 성적에 반영 됌.
   3. 게임 승리시 클럽 등급 올라감. (상대팀 등급에 비례)
6. 게임 신청(생성된 게임에 요청)
7. 게임 수락(게임 생성한팀이 상대팀 정보보고 수락/거절)
8. 게임 내 스탯 저장
   - 1점(자유투), 2점, 3점, 리바운드, 어시스트, 스틸, 파울
----------
## 기술 스택
- Java 17, Spring Boot 3.2.3
- H2, Redis
- JPA, QueryDSL
- Thymeleaf, JavaScript
